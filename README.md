
# GeoVizPy: Geophysical data visualization with PyVista and Trame

https://gitlab.in2p3.fr/spasquet/geovizpy

## Library requirements :
To use GeoVizPy, the following python packages are required :

- trame
- trame-vuetify
- trame-vtk
- trame-components
- vtk
- pyvista
- numpy
- scipy
- rasterio
- owslig
- gemgis
- fiona
- rioxarray
- scikit-image
